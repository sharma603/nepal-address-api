🇳🇵 Nepal Address API

A clean, scalable, GitHub-only Static API system.

Provinces

List of all provinces in Nepal.

GET /api/v1/provinces.json

Districts

Districts split by province for performance.

GET /api/v1/districts/province-{id}.json

Municipalities

Municipalities filtered by district.

GET /api/v1/municipalities/district-{name}.json

Wards

Ward-level data for metropolitan cities.

GET /api/v1/wards/{city}.json

Full Dataset

The complete aggregated address data.

GET /api/v1/full/nepal-full.json

Search Index

Optimized files for global search.

GET /api/v1/search/districts.json
